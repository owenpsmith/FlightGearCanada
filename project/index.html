Introduction

FlightGear is an open source flight simulator. My project website is FlightGear Canada. This is one of several regional portals in the flightgear community. The general goal of each is to provide a site where local users and developers can share, discuss and download local subjects. I took over as the administrator for FlightGear Canada in January. Since then I have been redeveloping the site from scratch to replace the existing bare bones site, using skills learned in this course and previous courses.

Credits

This website was developed entirely by myself, within the timeline of this course. The site contains content (3D models, models textures, and thumbnails) developed by other members of the FlightGear community. It also makes use of the following libraries:

  • Hangar - Renders AC3D format 3D models on HTML canvas element
